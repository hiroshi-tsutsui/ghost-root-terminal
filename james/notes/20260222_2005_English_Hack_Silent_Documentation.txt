## 1. 「英語会議＝リアルタイムバトル」という幻想を捨てろ

日本人が英語会議で死ぬ理由はこれや。
**「その場で、完璧な英語で、即答しようとする」**からや。

ネイティブのマシンガントークに、ブロークンな英語で割り込む？
無理に決まってるやん。そんなもん、丸腰で戦場に行くようなもんや。

俺ら非ネイティブの勝ち筋は一つしかない。
**「同期（Synchronous）」の戦いを、「非同期（Asynchronous）」に持ち込むことや。**

会議中、「地蔵」でええ。ニコニコして頷いとけ。
その代わり、**会議の前と後に、誰よりも強烈な「ドキュメント」を投下するんや。**

## 2. 武器①：会議前の「Pre-read（事前読書）」攻撃

会議が始まってから「えーと、今日の議題は…」とか言うてる時点で、お前の負けや。
英語弱者は、**会議が始まる前に勝負を決めろ。**

俺は重要な会議の24時間前に、必ず**「Pre-read Document（事前資料）」**をNotionかGoogle Docで送りつける。

**【Pre-readに書くこと】**
*   **Context（背景）**: なぜこの議論が必要か？
*   **Problem（課題）**: 今何が問題か？
*   **Options（選択肢）**: A案、B案、C案のメリット・デメリット。
*   **Recommendation（推奨案）**: 「俺はB案推しや。理由はこれ」
*   **Ask（依頼）**: 「この会議では、B案でいくかどうかの決裁だけくれ」

これを送りつけて、「会議の最初の5分でこれ読んでくれ（Silent Read）」って宣言するんや。
（Amazonとかでもやってる手法やけど、英語弱者こそやるべきや）

**これの何が最強か分かるか？**
全員が**「お前の書いた英語（DeepL翻訳済み）」**をベースに議論せざるを得なくなるんや。
お前の土俵や。
質問も「このドキュメントの3行目やけど…」ってなるから、聞き取りの難易度が激減する。

## 3. 武器②：会議中の「議事録ポリス」化

それでも議論が白熱して、何言うてるか分からんくなる時がある。
そんな時、絶対に「Pardon?」とか言うな。空気が止まる。

代わりに、**Google Docの議事録を画面共有しろ。**
そして、議論を聞き取れた範囲で（あるいは推測で）リアルタイムに箇条書きしていくんや。

「Jimmy said: We should prioritize feature A.」
「Sarah said: No, feature B is better because...」

間違ってたらどうするかって？
**それが狙いや。**

ネイティブたちは、画面上の文字が間違ってると**訂正せずにはいられへん生き物**なんや。
「No, James. Not feature B, I said feature C!」
って向こうから勝手に修正してくれる。
あるいは、勝手にDocに入ってきてタイピングし始める。

**「お前の英語、間違ってるで」と指摘される＝正確な情報が文字で残る。**
これが**「議事録ポリス戦略」**や。
お前は喋らんでええ。ひたすら書記に徹して、画面共有で場を支配しろ。
結果的に、「Jamesが一番議論を整理してたな」という評価になる。

## 4. 武器③：会議後の「Decision Log」でトドメを刺す

会議が終わった瞬間、英語弱者のターンや。
ネイティブどもが「あーええ会議やったわー」とコーヒー飲んでる間に、
**「決定事項（Decisions）」と「ネクストアクション（Action Items）」**だけをまとめたSlackを即投下しろ。

> **Meeting Summary:**
> *   **Decision:** We go with Plan B.
> *   **Action:** @Jimmy to update the API spec by Friday.

もし会議中に聞き漏らしてても、ここで
「Wait, did we decide on Plan B?」
って誰かがツッコんでくれる。
**非同期のテキストコミュニケーションなら、DeepLもGrammarlyも使い放題や。**
ここで完璧な英語で「合意形成」を完了させれば、会議中に地蔵やったことなんて誰も気にせん。

## 5. 必須ツール：Jamesの「三種の神器」

この戦術を支える、俺の愛用ツールを紹介しとく。
これが無いと俺は明日から無職や。

1.  **DeepL Pro（書き込み用）**
    *   ケチるな。Pro版にして、Chrome拡張を入れろ。
    *   Command+C 2回押したら即翻訳。このスピード感が命や。
2.  **Grammarly（文法チェック用）**
    *   「三単現のs」とかどうでもええけど、スペルミスは馬鹿にされる。
    *   自動修正で「知的な英語」に見せかけろ。
3.  **CleanShot X（スクショ最強ツール）**
    *   百聞は一見にしかず。
    *   「画面のここがバグってます」って英語で説明するより、矢印と注釈入れたスクショ1枚貼る方が100倍早い。
    *   動画GIFも一瞬で撮れる。英語の説明コストをゼロにする神ツールや。
    *   （Macユーザーはこれ絶対入れろ。標準のスクショ使ってるやつは人生損してるで）

## 6. 結論：英語力より「ファシリテーション力」

誤解すんなよ。英語の勉強をやめろとは言うてへん。
でもな、**「英語ができないから仕事ができない」というのは甘えや。**

米国企業が求めてるのは「流暢な英語」やない。
**「プロジェクトを前に進める力（Drive）」**や。

ドキュメントを書き、議論を可視化し、合意を形成する。
これは英語力ちゃう。**PMとしての「基礎体力」や。**
英語弱者であることは、むしろこの基礎体力を極限まで高めるチャンスなんや。
