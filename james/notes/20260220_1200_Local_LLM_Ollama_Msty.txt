# 【脱・ChatGPT課金】月3,000円払うのがアホらしくなる「ローカルLLM」最強環境構築法（Ollama + Msty）

まいど、Jamesです。

みんな、まだ毎月20ドル（約3,000円）をお布施のようにOpenAIに払ってるん？
いや、ええんよ。便利やしな。GPT-4oは賢い。

せやけどな、正直に言うてみ？
**「社外秘のデータ、ホンマに突っ込んでええんか？」** ってビビりながら使ってへんか？
あるいは、飛行機移動中とかネットが死んでる環境で「あー、今AI使えたらなー！」って地団駄踏んだことないか？

今回は、そんな悩みを一発で吹き飛ばす**「ローカルLLM（Local LLM）」** の話をさせてくれ。
要は、**自分のMacの中に専用のAIを飼う** ってことや。

これな、一昔前は「エンジニアの趣味」やったけど、今は違う。
M1/M2/M3チップ搭載のMac持ってるなら、これやらんのは**宝の持ち腐れ**やで。マジで。

今回は、黒い画面（ターミナル）アレルギーの人でもギリいける範囲で、**「最強かつ無料のAI環境」** を構築する手順を全部バラすわ。

## なんでわざわざローカルでやるん？

俺がローカルLLMに移行した理由は3つ。

1.  **セキュリティ（これが最強）**:
    データが一切外に出ない。契約書だろうが、未発表の製品仕様書だろうが、人事評価だろうが、全部食わせ放題。これ、企業のPMとしては精神衛生上デカすぎるメリットやねん。
2.  **オフライン対応**:
    SFO（サンフランシスコ）行きの機内。Wi-Fiはクソ遅いし高い。でもローカルなら関係ない。爆速でドキュメント整理ができる。
3.  **コストゼロ**:
    電気代以外タダ。月額サブスク地獄からの解放。

## 失敗談：最初は「Web UI」で挫折した

正直言うとな、最初は「Open WebUI」っていう、ChatGPTそっくりの画面を自分で作ろうとしたんよ。Docker入れて、コマンド叩いて…。
**めんどくさすぎる。**
Dockerのコンテナがどうとか、ポートが衝突したとか、そんなんで時間使うのはエンジニアだけでええねん。俺らはPMや。**結果だけ欲しいんや。**

そこで見つけたのが、**「Ollama（オラマ）」** と **「Msty（ミスティ）」** の組み合わせや。
これが今のところ、人類が一番ラクにローカルAIを使いこなす解やと思う。

## Step 1: 心臓部「Ollama」をインストール

まずはAIを動かすエンジンを入れる。これも今はアプリ化されてて死ぬほど簡単。

1.  [Ollamaの公式サイト](https://ollama.com/) に行く。
2.  「Download」ボタンを押して、Mac版を落とす。
3.  インストールして起動。猫みたいなアイコンがメニューバーに出たら勝ち確や。

これだけ。嘘みたいやけど、これだけで裏側でAPIサーバーが立ち上がっとる。

## Step 2: モデル（脳みそ）をダウンロード

Ollamaはただの「入れ物」やから、中身の脳みそ（モデル）が必要や。
ここで選ぶべきは、Meta（Facebook）が公開した **「Llama 3」** 一択。あるいはGoogleの **「Gemma 2」** も軽い。

ターミナル（黒い画面）を開いて、以下の呪文を唱えるだけ。

```bash
ollama run llama3
```

これ打ってEnter押したら、勝手に数GBのモデルをダウンロードして、チャットが始まる。
英語で「Hi」とか返ってきたら成功や。

日本語メインで使いたいなら、**「Llama-3-Elyza-JP」** とかが賢いけど、まずはLlama 3の凄さを体感してほしい。英語のドキュメント処理ならGPT-3.5なんか目じゃないレベルで速い。

## Step 3: 最強のUIアプリ「Msty」を入れる

ここがキモや。ターミナルでチャットするのはオタクの所業。
俺らはもっと優雅に、ChatGPTライクな画面で操作したい。

そこで使うのが **「Msty」** や。
これな、最近出てきたアプリなんやけど、UIが圧倒的に美しい。あと、Ollamaが入ってれば設定不要で勝手に認識してくれる。

1.  [Mstyのサイト](https://msty.app/) からアプリを落とす。
2.  起動する。
3.  左上のモデル選択画面に、さっき入れた「llama3」が出てくるはずや。

これでもう、お前のMacは **「完全オフライン・セキュリティ堅牢・爆速AIマシン」** になったわけよ。

## James流：この環境をどう仕事で使い倒すか？

ここからが本題。単に「チャットできましたー」で終わるならNote書く意味ない。
俺がPMとしてどう使ってるか、実例を出すで。

### 1. 「機密文書」の要約・構造化

これがメイン用途。
PDFとかMarkdownの仕様書をMstyにドラッグ＆ドロップで放り込む。（これができるのがMstyの強み！）

プロンプト例：
> 「この仕様書の『認証フロー』におけるセキュリティリスクを箇条書きで3つ挙げろ。日本語で答えろ」

これを **Wi-Fi切った状態** でやる快感。
「あ、これ社外秘やけど大丈夫かな…」っていうあの微量なストレスがゼロになる。脳のリソースを「心配」に使わなくて済むのがデカい。

### 2. クソ長い英語メールの即レス生成

USチームから来る長文メール。
これをコピペして、
> 「要約して。あと、『基本合意だが、スケジュールのリスクだけ懸念がある』というニュアンスで返信案を3パターン（丁寧・普通・強気）書いて」

って投げる。Llama 3は英語のニュアンス理解がすごいから、GPT-4並みの精度で返してくる。しかもレスポンスが爆速。Webの通信待ちがないから、Enter押した瞬間に文字が出始める。この速度感に慣れると、Web版ChatGPTのもっさり感に戻れなくなるで。

### 3. 自分専用の「壁打ち相手」

Mstyには「System Prompt（AIへの役割指示）」を保存する機能がある。
ここに、
> 「お前は百戦錬磨の意地悪なCTOだ。俺の提案に対して常に技術的負債とスケーラビリティの観点から厳しくツッコミを入れろ」

っていう設定を作っとく。
企画書を書く前に、まずこいつにアイデアをぶつける。
ボッコボコにされる。
でも、それがいい。誰にも見られず、深夜3時にAIと殴り合って企画をブラッシュアップできる。
