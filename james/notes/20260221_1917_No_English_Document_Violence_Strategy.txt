## 1. 日本人の「英語信仰」という呪い

まず、前提をひっくり返しましょう。
多くの日本人は「完璧な英語を話せるようになってから、海外に挑戦しよう」とします。

**これ、完全に間違いです。**
断言しますが、そんな日は一生来ません。

### 1-1. ネイティブと「口喧嘩」で勝とうとするな
我々が日本語で思考してる時点で、ネイティブとの「口頭レスポンス」の勝負には絶対に勝てません。
あいつら、呼吸するように議論します。割り込みます。早口です。

この土俵（Sync Communication）で戦おうとするから、「あぁ、自分はダメだ…」って心が折れるんです。
**土俵を変えるんです。**

### 1-2. 「同期」から「非同期」へ戦場を移せ
僕が徹底してるのは、**「口頭での意思決定を極限まで減らす」**こと。
その代わりに、**「非同期（Async）」なドキュメント**で圧倒的な質を見せつける。

米国企業、特にテック系は「合理的」です。
発音が綺麗なだけの無能より、**「テキストで論理的かつ実装可能な仕様書を書ける無口なやつ」**の方が100倍評価されます。

## 2. 実践：ドキュメント暴力（Visual Violence）

じゃあ具体的に何をするんか。
僕が呼んでる必殺技が**「ドキュメント暴力（Visual Violence）」**です。

相手が反論する隙間もないくらい、図解とロジックで埋め尽くされたドキュメントを、会議の「前」に送りつけるんです。

### 2-1. DeepL x Notion は「最強の武器」
僕のNotionの書き方はこうです。

1.  **日本語で爆速でロジックを組む**（思考の解像度を最大化）。
2.  **DeepL Pro（有料版）** で英語に変換。
3.  **Grammarly** でトーン＆マナーを補正。
4.  **CleanShot X** で撮ったスクショに矢印を入れまくって「視覚情報」だけで通じるようにする。

英語が苦手な人ほど、**テキストより「図」に頼ってください。**
矢印（→）、枠囲み、番号振（①②③）。これらは世界共通言語です。
「This button triggers this event」って書くより、**ボタンから矢印引いてイベントのログ画面貼る方が、誤解がないんです。**

### 2-2. 会議は「確認の場」にする
僕が主催する会議では、冒頭にこう言います。

> *"My English is not perfect, so I wrote everything down in this document. Please read the 'Decision Required' section first."*
> （英語完璧ちゃうから全部ここに書いた。まずは「意思決定が必要な箇所」だけ読んでくれ。）

これで会議の主導権（Agenda）は完全にこっちのものです。
参加者は僕のドキュメントを読むしかない。
そして、そのドキュメントが完璧なら、彼らは「Yes」と言うしかない。

**英語力で負けてても、準備力で勝つ。** これがジャパニーズ・ビジネスマンの生存戦略です。

## 3. 「議事録」こそが権力である

もう一つ、絶対やるべきハックがあります。
それは**「会議直後の議事録（Recap）爆撃」**です。

### 3-1. 歴史は「書いたやつ」が作る
会議中、ネイティブ同士が盛り上がって、なんか決まったような決まってないような空気で終わること、ありますよね？
ここで、**会議終了後5分以内**に、Slackにこう投稿するんです。

> **[Recap] Quick Summary**
> - **Decision:** We will use Library A instead of B.
> - **Action Item:** @Mike to implement the prototype by Friday.
> - **Context:** Library B has a latency issue (see link).

これを投稿した瞬間、**この内容が「事実」になります。**
もし会議中の理解が間違っててもええんです。間違ってたら誰かが「No, actually...」って訂正してくれます。訂正されなかったら、それが正義になる。

**「英語が聞き取れない」と黙って座ってるだけのやつは、会議にいないのと同じです。**
でも、**「間違っててもいいから要約を投げるやつ」は、チームの推進力として認知されます。**

## 4. ツールへの投資をケチるな

この「非同期＆ドキュメント戦略」を実行するために、僕はツールへの投資を惜しみません。
これが僕の**PMとしての武器（Arsenal）**です。

1.  **DeepL Pro**: 必須。セキュリティ的にも無料版は業務で使うな。
2.  **Grammarly Premium**: 「偉そうな英語」や「丁寧すぎる英語」を調整するために必要。
3.  **CleanShot X**: Mac最強のスクショツール。これがないと仕事にならん。注釈機能が神。
4.  **Raycast**: スニペット機能で「よく使う英語の定型文」を0.5秒で呼び出す。

「月額千円、二千円」をケチって、年収1000万のチャンスを逃すな。
プロなら道具に金をかけろ。これは**投資対効果（ROI）がバグるくらい高い投資**や。
