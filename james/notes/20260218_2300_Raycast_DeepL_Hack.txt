# 【脱・英語学習】米国PMが教える「Raycast × DeepL」の秒速レスポンス術（設定ファイル付き）

まいど、Jamesです。

今日は「英語の勉強」の話は一切せえへん。
**「英語力が低いまま、どうやってネイティブの議論に秒で食らいつくか」** という、もっと切実で実践的な話をします。

ぶっちゃけな、米国企業のPMやってて一番しんどいのは「時差」でも「文化」でもない。「レスポンスの速さ」や。
Slackで議論が始まった瞬間、もたもた翻訳サイト開いてコピペしてたら、もう話題変わっとるんよ。置いてけぼりや。

「英語勉強しなきゃ...」とか思うな。その時間はもうない。
代わりに、Macに **Raycast** 入れて、今から書く設定をそのままパクってください。
これで君の英語レスポンス速度は、TOEIC 900点の人より速くなる。

## 1. なぜ「Raycast」なんか？
Spotlightの代わり？ いやいや、そんな生ぬるいもんやない。
Raycastは **「キーボードから指を離さずに、DeepLとChatGPTをしばき倒すためのコマンドセンター」** や。

わいが実際にやってる設定はこれだけ。

1.  **DeepL翻訳:** `Option + D` で即起動
2.  **ChatGPT修正:** `Option + G` で文法・ニュアンス修正
3.  **スニペット:** `Cmd + Shift + V` で定型文呼び出し

マウスでカチカチやってる暇があったら、キーボード叩け。

## 2. 【即効】DeepL連携のセットアップ（所要時間: 3分）

### Step 1: Raycastを入れる
まず [Raycast公式サイト](https://www.raycast.com/) から落としてインストール。無料や。
Spotlightのショートカット（`Cmd + Space`）を無効化して、Raycastに割り当てるのが基本やけど、そこは好みでええ。

### Step 2: "DeepL Translate" Extensionを入れる
Raycastを開いて（`Cmd + Space`）、`Store` と打つ。
検索窓で `DeepL` を探して `Install Extension`。

### Step 3: API Keyを設定する（ここがミソ）
無料版のDeepLでもええけど、仕事で使うなら **DeepL API Free**（月50万文字まで無料）に登録して、API Keyを取得することをおすすめする。
これでコピペの手間すらなくなる。

### Step 4: ホットキーを割り当てる（最重要）
ここが勝負の分かれ目や。
Raycastの設定画面 (`Cmd + ,`) → `Extensions` → `DeepL Translate` を開く。
`Translate Selection` というコマンドに、**`Option + D`** を割り当てろ。

**【使い方】**
1.  Slackやブラウザで、訳したい英語を選択する。
2.  **`Option + D`** を叩く。
3.  画面中央に「翻訳結果」がポップアップする。

**これだけや。**
いちいち `Cmd + C` して、ブラウザ開いて、DeepL開いて、`Cmd + V` してた「あの無駄な3秒」が「0.5秒」になる。
1日100回翻訳したら、これだけで5分の節約や。いや、時間の問題ちゃうな。「思考が途切れない」のがデカいんや。

## 3. 【禁断】ChatGPTで「ネイティブっぽい」返信を捏造する

翻訳しただけやと、なんかロボットっぽいよな？
そこでChatGPTの出番やけど、これもブラウザ開いたら負けや。

### Raycast AI (or ChatGPT Extension) を使う
Raycastには `Raycast AI`（有料）があるけど、無料で行きたいなら `ChatGPT` Extensionを入れる。
API Keyを設定したら、以下のプロンプトを **AI Commands** として登録するんや。

**Command Name:** `Fix Grammar & Tone`
**Prompt:**
```
以下の英文を、ビジネスチャット（Slack）に適した、自然で簡潔な英語に修正してください。
元の意味を変えず、ネイティブのPMが使うようなトーン（少しカジュアルだが失礼ではない）にしてください。

Input: {selection}
```

これを **`Option + G`** に割り当てる。

**【実戦の流れ】**
1.  自分で適当な英語（Broken EnglishでOK）を打つ。
    *   例: `i think this ui is bad, user cannot find button.`
2.  その文字を選択して **`Option + G`**。
3.  AIが修正案を吐く:
    *   `I'm concerned about this UI; users might struggle to locate the button.`
4.  `Enter` ターン！ 送信！

どうや？ 
相手からしたら「こいつ、英語めちゃくちゃ自然で速いな...」ってなる。
中身はAIやけどな。結果が出ればええんよ、仕事なんやから。

## 4. 失敗談：スニペットをナメてた頃の話

昔のわいは、毎回毎回「Thank you for your update. regarding the ticket...」とか手打ちしてた。
アホや。腱鞘炎になるわ。

Raycastの **Snippets** 機能を使え。
わいが登録してる「神スニペット」3選を置いておく。

| Keyword | Content (展開される文章) | 用途 |
| :--- | :--- | :--- |
| `;thx` | `Thanks for the update!` | 基本の返信 |
| `;lgtm` | `Looks good to me. Please proceed.` | 承認・GOサイン |
| `;sch` | `Let me check my calendar and get back to you.` | 日程調整の逃げ |

キーワード（Trigger）は `;` セミコロン始まりにするのがコツや。普通のタイピングと被らんからな。
これ登録しておくだけで、Slackの返信速度が3倍になる。

## まとめ：道具を磨け、自分を磨くな

精神論で「英語頑張ろう」とか言うてる暇があったら、今すぐRaycast入れてショートカット設定してくれ。

1.  **`Option + D`** で読む（DeepL）
2.  **`Option + G`** で書く（ChatGPT）
3.  **`;xxx`** で定型文（Snippets）

この3つの武器だけで、君は「英語ができない日本人」から「レスが爆速な仕事できる人（英語は知らんけど）」にジョブチェンジできる。
