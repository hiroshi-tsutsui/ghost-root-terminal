## Espansoとは何か？

一言で言うと、**「クロスプラットフォーム対応・Rust製のテキスト展開ツール」** です。

- **Windows, Mac, Linux 全部動く**（これ重要。OS変わっても設定使い回せる）
- **設定はすべてYAMLファイル**（Gitで管理できる！）
- **動的な処理が可能**（PythonやShellスクリプトの結果を展開できる）
- **爆速**（Rust製だから）

要は、`:date` と打てば `2026-02-20` に変換されるアレです。
でも、Espansoはただの変換ツールじゃありません。**「魔法」** です。

Homebrew一発です。

```bash

```

## James流「神YAML」設定例

ここからは、実際に僕が毎日使い倒している `match.yml` の中身を晒します。
コピペして使ってください。**マジで世界変わるから。**

設定ファイルは通常ここにあります：
`~/Library/Application Support/espanso/match/base.yml`

### 1. 基本の「日付」系（動的展開）

PMやってると、ファイル名に日付入れること多くないですか？

```yaml
  # 日付展開
  - trigger: ":date"
    replace: "{{mydate}}"
    vars:
      - name: mydate
        type: date
        params:
          format: "%Y-%m-%d"

  # タイムスタンプ（ユニークIDとして使う）
  - trigger: ":ts"
    replace: "{{mytime}}"
    vars:
      - name: mytime
        type: date
        params:
          format: "%Y%m%d_%H%M"
```

`:date` → `2026-02-20`
`:ts` → `20260220_1345`

これだけでファイル保存時の「えーっと今日何日だっけ？」が消滅します。0.5秒の短縮ですが、一生で換算すると3日分くらい得します。

### 2. GitHub / Jira 用の定型文（カーソル位置指定）

これが最強。
レビュー依頼やバグ報告のテンプレを一瞬で呼び出し、**カーソルを書きたい場所に自動移動**させます。
`$|$` がカーソル位置です。

```yaml
  # GitHub PR Template
  - trigger: ":ghpr"
    replace: |
      ## 概要
      $|$

      ## 変更点
      - 

      ## 関連Issue
      - #
```

`:ghpr` と打つだけで、フォーマットがドン！と出て、カーソルは「概要」の下にスタンバイ。
マウスに手を伸ばす必要ゼロ。

### 3. シェルスクリプト連携（IPアドレス取得）

ここからがEspansoの変態的なところ。
**シェルコマンドの実行結果**を展開できます。

```yaml
  # パブリックIPを取得
  - trigger: ":myip"
    replace: "{{ip}}"
    vars:
      - name: ip
        type: shell
        params:
          cmd: "curl -s ifconfig.me"
```

`:myip` と打つと、一瞬のラグの後、自分のグローバルIPが入力されます。
インフラ作業やFW設定の申請時に「あれ、今のIPなんだっけ...」ってブラウザ開かなくていいんです。神か。

### 4. フォーム機能（定型メール）

変数が複数ある定型文（相手の名前とか日付とか）も、Espansoならフォームが出せます。

```yaml
  # ミーティング調整メール
  - trigger: ":mtg"
    form: |
      [[name]] 様
      
      Jamesです。以下の日程でいかがでしょうか？
      
      候補1: [[date1]] 10:00-11:00
      候補2: [[date2]] 14:00-15:00
      
      Zoom: https://zoom.us/j/123456789
      
      よろしくお願いいたします。
```

`:mtg` と打つと、GUIのポップアップが出て、「name」「date1」「date2」を入力してEnter押すと、完璧なメールが生成されます。
脳死でメールが打てます。

## 設定をGitで管理する（完全勝利）

ここが重要です。
`espanso` ディレクトリごとGitリポジトリにして、GitHubのプライベートリポジトリにPushしておきましょう。

```bash
cd ~/Library/Application\ Support/espanso
git init
git add .
git commit -m "Initial commit"
# リモートリポジトリに追加...
```

これで、新しいMacを買っても、`git clone` して `espanso restart` するだけ。
**「俺の最強の辞書環境」が、いつでもどこでも再現されます。**
