# 【就活全否定】ES手書き？ 御社が第一志望？ 茶番はもう終わりにせえ。

## 戦略分析メモ
- **ターゲット:** [A] 就活・ES・ガクチカに疲弊している学生
- **仮想敵:** 「就活は足で稼げ」「手書きの熱意」「みんなと同じリクルートスーツ」
- **Jamesの逆張り:** 「就活なんてAIにやらせて、自分は『個』を磨け。新卒カードなんて捨ててもええ。」

---

最近、カフェで就活生がES（エントリーシート）必死に書いてるの見かけるんよ。
しかも、まさかの**手書き**で。

**正気か？**

令和も8年（※2026年想定）になろうとしてんのに、まだそんなことやってんの？
「手書きの方が熱意が伝わる」とか言うてる人事おるらしいけど、そんな会社、入ってからが地獄やで。
非効率の塊みたいな場所で、君の大事な20代をドブに捨てることになる。

今日は、日本の「就活教」という宗教から、君らを解脱させる話をするで。

## 1. 「御社が第一志望です」という最大の嘘

みんな平気な顔して嘘つくよな。
「御社の理念に共感しました！」
「第一志望です！」

**嘘つけ。**
本当は「給料良さそう」「安定してそう」「親が喜ぶから」やろ？

その嘘をつくために、何十社も受けて、何十回も面接で頭下げて...。
それ、楽しいか？
自分の心殺してまで入った会社で、幸せになれるわけないやん。

俺がアメリカのテック企業に入った時、そんな嘘一切つかんかったで。
「このプロダクトが好きやから作りたい」
「俺のスキルはこれや。これだけ貢献できる。だからこれだけ金くれ」

これだけや。シンプルやろ？
**ビジネスは「価値交換」や。**
「御社への愛」とか「忠誠心」なんて、通貨にならへんのよ。

## 2. ガクチカなんて「捏造」でええ（良い意味で）

「学生時代に力を入れたこと（ガクチカ）」がない...って悩んでる奴多いな。
サークルの副代表とか、バイトリーダーとか、みんな似たり寄ったりの話ばっかり。

あのな、**企業が見てるのは「過去」ちゃう。「再現性」や。**
「こいつは課題に対してどうアプローチして、どう解決する人間か？」が見たいだけ。

やから、今から作ればええねん。
バイトの話を盛るんちゃうで？

**今すぐ、AI使って何か一つWebサービス作ってみ？**

「プログラミングできません」？
甘えんな。ChatGPTかClaudeにコード書かせたら一瞬や。
「◯◯という課題を解決するアプリのコード書いて」って投げるだけ。

それをGitHubに上げて、デプロイして、友達に使ってもらう。
これだけで「AIを活用して自力でプロダクト開発し、ユーザーフィードバックを得た経験」になるんや。

サークルの飲み会幹事の話より、よっぽど市場価値高いで。
**「過去の栄光」を探すんじゃなくて、「今」実績を作るんや。**

## 3. 【禁断のAI術】ES量産＆面接ハック

まともにES書いてたら時間がいくらあっても足らん。
AIに「君の人格」をインストールして、分身に書かせるんや。

### ES自動生成プロンプト（GPT-4 / Claude 3 Opus推奨）

まず、自分の「強み」「エピソード（箇条書きでOK）」「志望企業の求める人物像（HPからコピペ）」を用意してな。
ほんで、このプロンプトに突っ込むんや。

```markdown
# Role
あなたは世界一流のキャリアコンサルタントであり、私の専属ライターです。
論理的かつ情熱的な文章で、採用担当者の心を動かすESを作成してください。

# Input Data
- **私の強み:** [ここに強みを入れる。例：行動力、AI活用能力]
- **エピソード:** [ここに箇条書きでエピソードを入れる。例：AIを使ってアプリ開発した、英語学習コミュニティを作った]
- **志望企業の求める人物像:** [ここに企業の求める人物像をコピペ]
- **設問:** 「学生時代に最も力を入れたことは何ですか？（400文字）」

# Rules
- 結論から書く（PREP法）。
- 具体的な数字を入れる。
- 企業の「求める人物像」と私の「強み」がリンクするように構成する。
- 嘘は書かないが、表現は最大限魅力的にする。
- 文体は「です・ます」調で、自信に満ちたトーンにする。

# Output
最高品質の回答を3パターン作成してください（論理重視、情熱重視、バランス型）。
```

これで出てきた文章を、自分の言葉でちょっと直すだけ。
30分かかってたES作成が、3分で終わるで。
空いた27分で、映画見るなり、新しい技術触るなりした方がええやろ？

### 面接官を唸らせる「逆質問」リスト

面接の最後、「何か質問はありますか？」って聞かれるよな。
ここで「残業はありますか？」とか聞いてる時点で不合格や。

**逆質問は「アピールの場」や。**
「こいつ、視座が高いな...」と思わせる質問をぶつけるんや。

1.  **「御社の〇〇事業において、AI活用による業務効率化や新規サービス開発の余地はどのあたりにあると考えておられますか？（もし入社したらそこを担いたいです、という匂わせ）」**
2.  **「活躍している若手社員の方に共通するマインドセットや行動特性はありますか？（私はそれに合わせられます、という意思表示）」**
3.  **「御社が今後3年で最も注力するマーケットはどこですか？また、そこに対して新卒が貢献できる最大のポイントは何だとお考えですか？（経営視点持ってますアピール）」**

この3つを持っておけば、どの業界でも通用するで。

もっと気楽にいこうや。
