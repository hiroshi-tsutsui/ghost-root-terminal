# 【年収1500万】英語力ゼロの関西人が米国AI企業でPMやるための「生存戦略」全公開

こんにちは、Jamesです。
現在は米国の某AIテック企業でProduct Manager（PM）をやってます。

いきなりですが、結論から言います。

**「英語ができなくても、米国企業でPMはできる」**

これ、嘘やと思うやろ？
TOEIC 900点とか、ネイティブレベルの会話力が必要やと思ってる人が99%。
でもな、断言します。**それは「昭和のサラリーマン」の思考です。**

僕の英語スペック、聞きたい？
**「中1レベル」です。**
いや、マジで。Be動詞と一般動詞の使い分けが怪しいレベルからスタートしました。
それでも年収1500万（$100k+）の世界で、ネイティブのエンジニアやデザイナーを動かしてプロダクトを作ってる。

今日は、そんな僕がどうやって米国企業で生き残っているのか。
その**「泥臭すぎる生存戦略」**を全て公開します。
これは「英語の勉強法」ではありません。**「英語を使わずに勝つためのハック」**です。

## 1. なぜ「英語力」がなくても勝てるのか？

まず、大前提として知っておいてほしいこと。
**米国テック企業は「成果」しか見てへん。**

綺麗な発音？ 文法的に正しいメール？
そんなもん、**1円の価値もない。**
求められてるのは**「プロダクトを前に進めること」**だけ。

逆に言うと、英語がペラペラでも、
「何言ってるかわからん」「仕様が決まらん」「バグが減らん」PMは即クビです。
これ、日本企業とは決定的に違うとこ。

僕が生き残ってる理由はただ一つ。
**「英語以外の武器」を極限まで磨いたから。**
具体的には**「可視化（Visual）」と「非同期（Async）」**です。

## 2. 武器①：DeepLは「翻訳機」やない。「思考の整理機」や

みんなDeepL使ってると思うけど、使い方が甘い。
「日本語入れて、英語にする」だけやろ？
それじゃアカン。

僕の使い方はこう。

1. **日本語で論理をガチガチに固める。**
   - 箇条書き（Bullet points）で書く。
   - 主語と述語を明確にする。
   - 曖昧な表現（〜だと思われる、検討する）を排除する。
2. **DeepLに突っ込む。**
3. **出てきた英語を「再翻訳」して日本語に戻す。**
   - ここで意味がズレてたら、元の日本語が悪い。修正する。
4. **Grammarlyで文法チェック。**

これやるだけで、ネイティブよりも「論理的で分かりやすい英語」が爆誕します。
ネイティブは喋り言葉でダラダラ書くから、意外と何言ってるか分からんことが多い。
**「DeepL経由の英語」の方が、実はビジネスでは最強なんです。**

## 3. 武器②：CleanShot Xによる「視覚的暴力」

会議で英語が聞き取れへん？ 喋れへん？
**喋らなきゃいいんです。**

僕のPCには**「CleanShot X」**という最強のスクショツールが入ってます。
（Macユーザーなら絶対に入れろ。これがないと仕事にならん）

仕様を伝える時、僕はテキストなんて書きません。
1. 画面をスクショする。
2. CleanShot Xで矢印（Arrow）と枠（Rect）を引きまくる。
3. 手順番号（Counter）を振る。
4. **「Fix this like image.」** とだけ書いてSlackに投げる。

これだけ。
これだけで、インド人のエンジニアも、アメリカ人のデザイナーも、
**「Oh, understood.」** で動いてくれる。

言葉で「右上のボタンを赤くして、マージンを5px空けて...」とか説明するから事故るんです。
**「画像」は世界共通言語。**
言葉で戦おうとするな。**「画（え）」で殴れ。**

## 4. 武器③：Notionによる「ドキュメント駆動」

Zoom会議は僕にとって「地獄」です。
早口のネイティブ英語、飛び交うスラング、マイクのノイズ。
**あんなもん、聞き取れるわけがない。**

だから僕は、**「会議をさせない」**方向に全振りします。
そのための武器が**Notion**です。

1. 会議のアジェンダが決まった瞬間、Notionにページを作る。
2. 背景、目的、議論したい点、決定事項の案まで全部書く。
3. 会議の24時間前にSlackで共有。**「Read this before meeting. Comment if you have questions.」**
4. ネイティブたちがコメントで議論を始める。
5. 僕はDeepL片手にゆっくりコメントを返す。

会議が始まる頃には、**議論の8割は終わってるんです。**
会議は「決定」を確認するだけの場にする。
これなら、英語が聞き取れなくても「Yes/No」だけで乗り切れる。

これを**「非同期コミュニケーション（Asynchronous Communication）」**と呼びます。
米国テック企業では、むしろこっちの方が「仕事ができる奴」として評価される。
**「英語ができない」という弱点が、逆に「ドキュメント文化を定着させる」という強みに変わる瞬間です。**

## 5. 結論：英語コンプレックスを捨てろ

日本人は真面目すぎる。
「英語を勉強してから挑戦しよう」とか言うてる間に、
インド人や中国人は、もっと酷い英語でガンガン職を奪っていきます。

彼らにあって、日本人にないもの。
それは**「厚かましさ」**と**「道具を使う賢さ」**です。

英語なんて、所詮ツール。
AI翻訳もあれば、スクショツールもある。
**テクノロジーで補える欠点は、欠点じゃない。**

「英語ができないから...」と諦めてるエンジニア、PM志望の学生たちへ。
**今すぐその考えを捨ててください。**
君に必要なのは英会話スクールじゃない。
**CleanShot XとDeepL Proの契約です。**

そして、**「言葉を使わずに人を動かす技術」**を磨くこと。
これさえあれば、年収1500万の世界への扉は開かれます。
