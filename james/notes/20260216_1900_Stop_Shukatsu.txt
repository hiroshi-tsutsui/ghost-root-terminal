# 日本の「就活」で消耗してる暇あったら、さっさと「市場」変えて勝ちに行かんか？【年収1000万PMの生存戦略】

正直に言うていい？
いま、日本の「就活」ナビサイト眺めて、志望動機を捏造してる時間、**全部ムダ**やで。

いや、ムダって言うたら言い過ぎかもしれんけど、**コスパが悪すぎる**。

俺な、3年前までは偏差値なんぼかも知らん日本の大学で、単位ギリギリの凡人やってん。英語？ 「This is a pen」レベルや。
それが今、米国のテック企業でPM（プロダクトマネージャー）やって、完全フルリモートで、年収は日本円で1,000万超えてる。

別にこれ、俺が「天才」やったわけでも、裏口入学したわけでもない。
単に**「戦う場所」を変えただけ**やねん。

今日は、真面目な君らが陥りがちな「日本の就活・キャリアの罠」と、そこから抜け出して**「楽に、高く売れる」ルート**について、全部バラすわ。

## 1. 「御社の理念に共感しました」とか、誰も信じてへんて

まずな、日本の就活のエグいところは**「嘘つき大会」**になってるとこや。

学生は「御社が第一志望です！ 理念に感動しました！」って嘘つく。
企業側も「アットホームな職場です！ 若手が活躍！」って嘘つく。

これ、お互いに不幸やと思わん？

俺が今いる米国のテック業界（AI系）やと、面接で「理念への共感」なんて聞かれたこと一回もないわ。
聞かれるのはこれだけ。

**「君は、何ができるん？（What can you do?）」**
**「過去に何を作ったん？（What did you build?）」**

以上。

シンプルやろ？
ここに「熱意」とか「協調性」とか、計測不能なパラメータが入る余地はない。
コードが書けるか、プロダクトを回せるか。数字を作れるか。それだけや。

## 2. 「未経験歓迎」のSESに搾取されるな

エンジニア志望の子によくあるんが、「とりあえず未経験で育ててくれる会社に入って、スキルつけます！」ってやつ。

アカン、それ一番カモられるやつや。

「研修充実」とか書いてるけど、実際は現場に放り込まれて、エクセルのスクショ撮って貼り付けるだけの仕事させられる…なんて話、腐るほど聞くやろ？

俺がやったんは逆や。
**「先にスキルをつけて、実績を作ってから、自分を売り込む」**。

これ、難しそうに聞こえるかもしれんけど、今の時代めっちゃ簡単やで。

1. **作る:** UdemyとかYouTube見て、簡単なWebアプリ作る。（ToDoリストとかでええねん）
2. **公開する:** GitHubに上げる。動くURLを用意する。
3. **送る:** LinkedInで、興味ある会社のエンジニアとかリクルーターに直接DM送る。

日本の就活生がリクナビでポチポチしてる間に、俺はLinkedInで海外のスタートアップのCTOに直接、下手くそな英語でDM送ってた。

「Hey, I built this tool. It solves X problem. Check it out.」

これだけ。
100通送って、返事来たん3通や。
でも、そのうちの1社が「おもろいやん、話聞こうか」ってなって、今のキャリアに繋がってる。

**打率は低くてええ。バットを振る回数と、振る場所（市場）が大事なんや。**

## 3. 英語は「勉強」すな。「ツール」として使え

「英語できへんから無理です」って言う子多いけど、それも勘違いや。

俺も最初はTOEIC 400点台やった。
でもな、エンジニアとかPMの仕事って、使う単語決まってるねん。

- Deploy（展開する）
- Fix（直す）
- Issue（問題）
- Feature（機能）

これくらい覚えとけば、あとはDeepLとChatGPTがあれば仕事できる。
会議？ 今どきZoomの字幕機能あるし、最悪「Can you type it in chat?（チャットに書いて）」って言えばええ。

完璧な英語を話そうとするから動けへんねん。
**「ブロークンでも通じればOK、仕事が進めばOK」**っていうマインドセットが、外資で生き残るコツや。

## 4. 結論：日本の「常識」を捨てたもん勝ち

年収1,000万って、日本では「すごい人」かもしれんけど、シリコンバレーの感覚やと「新卒レベル」やったりする。

同じスキル、同じ労働時間でも、**「どこに身を置くか」で値段は何倍も変わる**。
マグロだって、近所のスーパーで売るより、豊洲の初競りに出した方が高いやろ？ それと一緒や。

君が今、就活で苦しんでるなら、それは君の能力が低いわけじゃない。
**売る場所を間違えてるだけ**かもしれん。

1. エントリーシート書くのをやめる。
2. GitHubのアカウント作る。
3. LinkedInのアカウント作る。
4. 英語アレルギーを捨てる。

この4つやるだけで、見える世界ガラッと変わるで。
騙されたと思って、今すぐやってみ。

ほな、また。
