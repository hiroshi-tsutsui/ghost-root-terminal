# 【保存版】時差16時間のチームで生き残るための「マウス禁止」生産性ハック（Raycast & Arc）

おつかれさまです。Jamesです。

今回は**「精神論」は一切ナシ**でいきます。
「モチベーションを上げよう」とか「朝活しよう」とか、そんなんはもうええねん。
我々が欲しいのは、**「今すぐ、確実に、楽をするための具体的な手順」**だけやろ？

僕は現在、アメリカ西海岸のチームとフルリモートで働いとるんやけど、時差が16時間もあるねん。
向こうが起きる頃にこっちは寝る。つまり、**「リアルタイムの会議」に頼ってたら死ぬ**環境や。
そこで編み出したのが、**「極限までキーボードから手を離さずに仕事を終わらせる」**ための環境構築。

今日は、僕が毎日使ってる「三種の神器」の中から、特に**Raycast**と**Arc Browser**の具体的な設定を全部晒すわ。これ真似するだけで、1日30分は絶対に浮くから、騙されたと思ってやってみて。

## 1. Raycast: MacのSpotlightは今すぐ捨てろ

まだSpotlight使ってる人おる？悪いこと言わんから、今すぐ**Raycast**に入れ替えてくれ。無料やし。
これはランチャーアプリなんやけど、僕にとっては**「第二の脳」**や。

### 失敗談：拡張機能を入れすぎてゴミ屋敷化
最初、僕はRaycastのStoreにある便利そうなExtensionを片っ端から入れたんや。「Spotify操作？入れる！」「GitHub連携？入れる！」ってな。
結果どうなったか？ **コマンドを探すのに時間がかかる**という本末転倒な事態に陥った。アホやろ。

### 解決策：エイリアス（Alias）を徹底的に仕込む
重要なのは「機能の数」やない。「呼び出す速度」や。
僕は設定画面（`cmd + ,`）のExtensionsタブで、以下のエイリアスを体に叩き込んでる。

| 機能 | エイリアス | 動作 |
| :--- | :--- | :--- |
| **Window Management** | `Left` / `Right` / `Max` | 画面分割を一瞬でやる。マウスでウィンドウ端をドラッグするとか、原始人のやることやで。 |
| **Clipboard History** | `cl` | クリップボード履歴。これが無いと生きていけへん。 |
| **Snippets** | `sn` | 定型文呼び出し。後述するけど、これが最強の時短。 |
| **My Schedule** | `sch` | 次の会議のZoom URLを1秒で開くためだけの設定。 |

**★ここがポイント:**
エイリアスは「2文字」が基本や。3文字打つ暇があったらコーヒー飲みたいからな。

### 最強機能：Snippets（スニペット）で「お世話になっております」を抹殺
メールやSlackで同じ文章を2回以上打ったら、それはもう「自動化のサイン」や。
RaycastのSnippets機能で、以下の登録をしてる。

- **`!mail`** → `お世話になっております。Jamesです。`
- **`!zoom`** → `こちらのリンクからお願いします: https://zoom.us/j/xxxxxxxxx`
- **`!rej`** → `申し訳ありませんが、現在はリソースが不足しておりお引き受けできません。またの機会にお願いいたします。`（断り文句も自動化や。心を痛める必要なし）

「辞書登録でええやん」って思った？
Raycastのスニペットは、**カーソル位置の指定**ができるんや（`{cursor}`って入れるだけ）。
`!mtg` → `お疲れ様です。次回のMTGですが、{cursor}でいかがでしょうか？`
って登録しておけば、展開した瞬間に日時の入力待ち状態になる。この「1秒の短縮」が、1日100回積み重なるとデカいんやで。

## 2. Arc Browser: 「タブを開きすぎる病」への処方箋

次はブラウザや。Chromeもええけど、僕は**Arc**に乗り換えた。
理由は単純。「タブを消すのが面倒くさい」からや。

### 失敗談：タブ100個開いてメモリ爆発
Chrome時代、僕は「あとで読む」タブを溜め込みすぎて、常にタブが米粒みたいになってた。
で、どれがJiraでどれがFigmaかわからんくなって、探すのにイライラする。この時間が一番ムダ。

### 解決策：Auto Archive（強制アーカイブ）機能
Arcには**「12時間（設定可）触らなかったタブを勝手に消す」**機能がある。
最初は「えっ、消されたら困るやん」って思ったけど、逆やった。
**「12時間触らんかった情報は、今の自分には不要」**なんや。
もし必要なら、履歴から探せばいい。この機能のおかげで、僕のブラウザは常にスッキリしてて、脳のメモリも解放された気分や。

### Spacesで「仕事」と「プライベート」を脳内分離
ArcのSpaces機能で、以下のように使い分けてる。

1.  **Space 1: Work (US)** - Jira, Confluence, Slack, GitHub
2.  **Space 2: Learning** - 英語記事, 技術ドキュメント, Udemy
3.  **Space 3: Private** - YouTube, Amazon, 旅行の計画

これをトラックパッドの2本指スワイプ（`ctrl + 数字`でも可）で切り替える。
仕事中にYouTubeの誘惑が目に入らんのがデカい。
**「物理的に見えなくする」**。これが意志の弱い僕らが集中するための唯一の解や。

## 3. 最後に：ツールに使われるな

ここまで偉そうに語ってきたけど、最後に一番大事なこと言うで。
**「設定に凝りすぎるな」**。

僕も昔は、Notionのダッシュボードをピカピカに作り込んで、それで満足して仕事した気になってた時期があった（黒歴史や）。
でも、ツールはあくまで「楽をするための道具」や。
**設定にかける時間は、週末の1時間だけ**って決めてる。それ以上やると、手段が目的化するからな。

まずは今日紹介した、
1.  Raycastを入れて、ウィンドウ整理のショートカットを設定する
2.  よく使う定型文を3つだけスニペット登録する

これだけでええ。今日からやってみて。
浮いた時間で何するかって？
サウナ行くに決まっとるやろ！

ほな、また。
