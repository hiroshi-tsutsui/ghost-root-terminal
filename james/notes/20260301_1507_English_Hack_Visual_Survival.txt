# 米国AI企業PMが教える「英語力0」の生存戦略：DeepLとCleanShot Xで殴り勝て

「英語ができないから、外資系企業なんて無理」
「TOEIC 900点取ってから転職活動します」

はっきり言うて、その思考停止が一番の機会損失や。

今すぐその参考書を閉じて、DeepLとスクリーンショットツールを立ち上げてください。

僕は現在、米国のAIスタートアップでPM（プロダクトマネージャー）として働いてます。年収は日本円で1000万を超えてるけど、正直に言うて、僕の英語力は「中学レベル」に毛が生えた程度です。ネイティブの高速マシンガントークなんて、半分も聞き取れてへん自信があります。

それでも、僕は「仕事ができるPM」として評価されてる。なぜか？

それは、**「英語力」という土俵で勝負するのをやめたから**です。

今日は、純ドメ日本人が米国企業で生存するための、泥臭くも最強の「非同期コミュニケーションハック」を全部公開します。これを読めば、「英語コンプレックス」なんてものが、いかに些末な問題やったかがわかるはずです。

## 1. 英語は「会話」するな、「ログ」を残せ

まず大前提として、僕らノンネイティブが、リアルタイムの口頭議論でネイティブに勝てるわけがないんです。

会議で急に話を振られて、「あー、うー」と詰まってる間に、話題は次のトピックへ。結果、「あいつは意見がないやつだ」とレッテルを貼られる。これが最悪のパターン。

じゃあどうするか？

**「戦場をテキスト（非同期）に移す」** んです。

### 必殺技：会議後の「即レス議事録」

僕は会議中、必死に聞き取るふりをしながら、実は議事録（Summary）の構成を考えてます。そして会議が終わった瞬間に、Slackにこう投げるんです。

> **"Thanks for the meeting! Here is a quick recap based on my understanding. Let me know if I missed anything."**
>
> **Decisions:**
> - feature A will be released next week.
> - B is postponed.
>
> **Action Items:**
> - @James: Create PRD for feature A (Due: tomorrow)
> - @Mike: Check API latency.

これが僕の生存戦略の第一歩。

口頭で曖昧だった部分も、テキストにしてしまえばこっちのもの。もし認識が間違っていれば、テキスト上で訂正が入るだけ。これが「英語ができない」ことの免罪符になります。

「あいつは会議で喋らないけど、まとめが早いし正確だ」

こう思わせたら勝ちです。英語力なんて誰も気にしなくなります。

## 2. DeepLは「翻訳機」やない、「推敲パートナー」や

「翻訳アプリ使うなんてダサい」と思ってませんか？ それ、ただのプライドです。捨ててください。

僕のPC画面の右側には、常にDeepLが鎮座してます。Slackを打つときは、まず日本語でバーッと書いて、DeepLに放り込む。

でも、ここで一つコツがあります。**「逆翻訳」を必ず確認すること**。

1. 日本語 → 英語
2. 出てきた英語 → 日本語（逆翻訳）

この逆翻訳された日本語が、元のニュアンスとズレていないかを確認するんです。もしズレていたら、元の日本語を「論理的でシンプルな文章」に修正する。これを繰り返すことで、DeepLが出力する英語の精度が劇的に上がります。

結果的に、僕がSlackに投稿する英語は、ネイティブも驚くほど「洗練されたビジネス英語」になってます。

DeepLはただの翻訳機ちゃいます。自分の思考を論理構造化するための「壁打ち相手」なんです。

## 3. CleanShot X：言葉の壁を「画像」で破壊する

ここからが本題。僕の最強の武器を紹介します。それが **「CleanShot X」** です。

Macユーザーでこれ使ってない人、正直人生損してます。

PMの仕事は「仕様を伝えること」や「バグを報告すること」が多いんですが、これを拙い英語で説明しようとするから事故るんです。

「The button on the top right corner overlaps with the header when scrolled...」

こんなん書いてる暇があったら、**スクショ撮って、矢印引いて、"Fix this" って書いた画像を送る**。これで1秒で伝わります。

### 「百聞は一見に如かず」を地で行く

CleanShot Xが最強なのは、以下の機能があるからです。

*   **Annotate（注釈）**: 撮った瞬間に矢印やテキスト、モザイクを入れられる。
*   **Scrolling Capture**: 長いWebページも一枚の画像にできる。
*   **GIF/Video**: 操作手順をその場で録画して、リンクとして共有できる。

特に「GIF録画」は神です。バグの再現手順なんて、言葉で説明するのは苦行でしかない。でも、GIFで見せれば一発です。エンジニアも「Oh, I see. Thanks based James.」で終わり。

英語の説明コストをゼロにする。これが「視覚情報の力」です。

僕のSlack投稿の半分は、CleanShot Xで生成された画像か動画リンクです。これのおかげで、「Jamesの仕様書は分かりやすい」と評価されてます。英語力関係ないやん、って話です。

## 4. 結論：ツールを使い倒す「度胸」がすべて

結局、米国企業で働くのに必要なのは、ネイティブレベルの英語力ちゃいます。

**「自分の意志を、手段を選ばず相手に伝える執念」** です。

DeepLを使おうが、スクショを送りつけようが、仕事が進めばそれが正義。逆に、綺麗な英語を話そうとしてモジモジしてる方が、プロとしては失格です。

テクノロジーは、僕らのような「持たざる者」のためにある。

もしあなたが「英語ができない」と悩んでるなら、今すぐDeepLをショートカットキーに登録して、CleanShot Xを導入してください。そして、泥臭くテキストと画像でコミュニケーションを取ってみてください。

世界は意外と、そんな「非ネイティブの生存戦略」を受け入れてくれます。

ほな、また。
