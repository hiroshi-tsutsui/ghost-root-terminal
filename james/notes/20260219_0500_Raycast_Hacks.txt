# マウス触ったら負けや。米国PMが生き残るための「Raycast」魔改造ガイド【コピペで終わる設定付き】

まいど、Jamesです。

今日はな、精神論は一切抜きや。「意識を高く持とう」とか「パッションが大事」とか、そんなんで飯は食えへん。特に時差16時間のチームとやり合いながら、英語のドキュメントを大量に捌かなあかん今の環境ではな。

必要なのは**「物理的な速度」**だけや。

今回は、僕が実際に使ってて「これ無かったら仕事辞めるわ」ってレベルの神ツール**「Raycast」**について、明日から使える具体的な設定だけを叩き込むで。

「え、Spotlightでよくない？」
「Alfred課金してるし…」

アカン。甘い。チョコモナカジャンボの真ん中くらい甘い。
Raycastはランチャーちゃうねん。**「指先で操るコクピット」**や。

この記事読んだら、多分マウス触る回数が1日500回は減るはずや。ほな、いくで。

## 1. なんでRaycastなん？（結論：DeepLと会議が一瞬で終わるから）

僕がRaycastに完全移行した理由はシンプル。
**「アプリの切り替え」という無駄な時間を0秒にしたかったから**や。

### 以前の僕（Before）
1. Slackで英語のメッセージ来る。
2. マウスでテキスト選択してコピー。
3. `Cmd + Tab` でChromeに切り替え。
4. ブックマークからDeepL開く。
5. ペーストして翻訳。
6. 「あ、これちゃうわ」ってなって修正してコピー。
7. Slackに戻ってペースト。

…書いててイライラしてきたわ。これ、1回30秒かかるとして、1日50回やったら**25分の無駄**や。週で2時間。月で8時間。1営業日分、ただのコピペに消えてる計算や。アホらしいやろ？

### 今の僕（After）
1. Slackで英語のメッセージ来る。選択してコピー。
2. `Option + Space` (Raycast起動)
3. `dl` と打ってEnter。
4. **その場で翻訳結果が出る。** `Enter` で翻訳文をコピー。
5. ペースト。

所要時間、**3秒**。
この「ウインドウを行ったり来たりしない」快感を知ったら、もう戻れへん。

## 2. 導入編：まずはこれだけ入れろ

インストールは公式サイトから `brew install --cask raycast` でもええし、普通に落としてもええ。
起動したら、まずは `Extensions` (拡張機能) ストアに行くんや。

僕が「これだけは入れとけ」って断言する**三種の神器**がこれや。

### ① DeepL Translate (必須ランク：SSS)
さっき言ったやつな。
APIキー設定とか面倒くさいこと言わん。無料版のWebスクレイピング的な挙動でも十分速いけど、Pro契約してるならAPIキー入れた方が安定する。

**James流設定：**
- Alias（ショートカット名）を **`dl`** に設定。
- これで `Option + Space` → `dl` → `Enter` だけで翻訳窓が開く。
- **失敗談：** 最初、Alias設定せんと毎回「DeepL」って打ってたんやけど、タイプミスして「DeepK」とかになってイラッとしてた。**エイリアスは2文字**。これ鉄則な。

### ② Google Translate (DeepLの補欠ちゃうで)
「DeepLあるやん」思うやろ？ちゃうねん。
Google翻訳の強みは「言語検出」と「発音」や。
英語のMTG前、どうしても発音が怪しい単語がある時。
`gt` → 単語入力 → その場で発音再生。
これで「ア、アイ・シンク…」ってどもらずに済む。

### ③ Clipboard History (標準機能やけど神)
これは拡張機能ちゃうけど、Raycastの標準機能で一番ヤバいやつ。
過去にコピーしたテキスト、画像、リンク、全部覚えててくれる。

**James流ハック：**
- ホットキーを `Cmd + Shift + V` に変更。
- メールアドレス、Zoomのリンク、定型文。全部ここから呼び出す。
- **「あれ、さっきコピーしたやつどこいった？」が人生から消滅する。**
- ちなみに、画像をコピー履歴から直接ドラッグ＆ドロップできる機能がついた時、僕は感動で泣いた（嘘やけど）。

## 3. 上級編：Script Commandsで「自分専用ボタン」を作る

ここからが本番や。Raycastは自分でスクリプト（Bash, Python, Node.js）を書いてボタンにできる。
「プログラミングとか無理」って？ 安心せえ。**ChatGPTに書かせたらええねん。**

僕が実際に作って毎日使ってるスクリプトを紹介するわ。

### 「Zoom爆撃」スクリプト
USとのMTG、開始1分前に「リンクどこやっけ！？」ってカレンダー漁るの、ダサいやろ？
このスクリプトは、**「直近のGoogleカレンダーの予定からZoom/Meetのリンクを抽出して開く」**だけのボタンや。

**作り方（ChatGPTに投げるプロンプト）：**
> 「MacOSのターミナルコマンドで、Googleカレンダーの直近の予定を取得して、もしZoomかGoogle MeetのURLが含まれてたら、それをデフォルトブラウザで開くシェルスクリプト書いて。RaycastのScript Command用で。」

これで作ったスクリプトを `join` って名前で保存。
会議の時間になったら `Option + Space` → `join`。
**一瞬で会議室イン。**
「Hey James, you're early.」
「せやな（ドヤ顔）」

## 4. 失敗から学んだ「やりすぎ」の罠

実は僕、最初はRaycastに**何でもかんでも詰め込みすぎて失敗した**んや。
Spotifyの操作、Jiraのチケット管理、GitHubのPR確認…全部Raycastでやろうとした。

**結果：**
ショートカット覚えきれんくて、結局マウスでポチポチ探すハメに。本末転倒や。

**教訓：**
**「1日10回以上やる作業」だけをRaycastに入れる。**
それ以外は普通にアプリ開いた方が早い。
まずは **「翻訳 (dl)」** と **「クリップボード履歴」** だけ。
この2つだけで、君の仕事の速度は確実に変わる。

## 5. まとめ：明日からのアクション

長々と語ったけど、やることは3つだけや。

1. **Raycastをインストールする。** (無料やし、迷う暇あったら入れろ)
2. **DeepL拡張機能を入れて、Aliasを `dl` にする。**
3. **マウスをデスクの遠くに置いて、なるべく触らんように仕事してみる。**

最初は「キーボードだけで操作するのムズイわ！」ってなるかもしれん。
でもな、自転車と一緒や。一度乗れるようになったら、歩くのがバカらしくなる。
僕らが目指してるのは、そういう「不可逆な進化」や。

ほな、設定終わったらSlackでドヤ顔してきてな。
Good luck.
