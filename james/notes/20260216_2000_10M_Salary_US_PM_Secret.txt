# 【年収1000万】凡人が「日本の就活」を捨てて米国テック企業のPMになった「裏ルート」全部話すわ

おい、そこの就活で死にかけてる学生、あと転職考えてるエンジニア。

毎日毎日、真っ黒なスーツ着て、謎のマナー講座受けて、「御社の理念に共感しました！」とか嘘ついてへんか？
ES（エントリーシート）何十枚も手書きして、祈られまくって、「自分は社会に必要ないんか…」って病みかけてるんちゃうか？

正直に言うわ。
**それ、全部無駄やで。**

今の日本の就活システムは、優秀な奴を安く買い叩くための「無理ゲー」や。
真面目にルール守って戦ってる時点で、お前は負け戦に参加させられてるんや。

俺も3年前はそうやった。
日本のFラン大学で単位ギリギリ、英語なんて「Hello」しか言えへんかった凡人や。
でも今は、**米国のテック企業（AI系）でPM（プロダクトマネージャー）やってる。**
**フルリモートで日本に住みながら、年収は1000万超えとる。**

「天才やったんやろ？」
「帰国子女なんやろ？」

ちゃうちゃう。俺はただ、**「戦う場所」と「戦い方」を変えただけ**や。
努力の方向性を変えるだけで、年収も働きやすさも劇的に変わる。

今日はその「裏ルート」、包み隠さず全部話したるわ。
綺麗事は一切抜きや。現実を見ろ。

## 1. 日本の「就活」という名の宗教

まず、お前らが苦しんでる原因を教えたる。
それは**「新卒一括採用」**という名の宗教や。

- **ポテンシャル採用:** スキルじゃなくて「素直さ」「若さ」を見られる。
- **謎のマナー:** ノックの回数とか、ハンコのお辞儀とか、本質と関係ないことで減点される。
- **総合職:** 入社するまで配属ガチャ。エンジニア希望なのに営業に回されるリスク。

これ、海外から見たら異常やで？
シリコンバレーで「御社の理念に共感しました！」なんて言うたら、「で、お前は何ができるん？」で終了や。

俺は日本で就活してた時、30社受けて全部落ちた。
「協調性がない」「個性が強すぎる」って言われてな。
でも、その「個性」こそが、海外では最強の武器になったんや。

## 2. 俺がやった「3つの戦略」

じゃあ、具体的にどうやって凡人の俺が米国企業のPMになれたんか。
やったことはシンプルや。

### ① 「英語」を勉強するな、「英語で」勉強しろ

「英語ができないから海外無理」って言う奴多いけど、それは言い訳や。
俺も最初はTOEIC 400点台やった。

でもな、**エンジニアやPMの仕事で使う英語なんて、パターン決まっとる**んや。
- `Fix bug` (バグ直せ)
- `Deploy to prod` (本番環境にあげろ)
- `LGTM` (ええやん)

これだけで会話成立するで（極論な）。
俺はインド留学に行ったけど、そこで学んだのは「綺麗な英語」やない。「通じればいい英語」や。
インド人の英語、聞いたことあるか？ めちゃくちゃ訛ってるし、文法も適当や。
でも彼らはGoogleやMicrosoftのCEOやっとる。なぜか？ **中身があるから**や。

**TOEICの勉強なんか捨てろ。**
GitHubのREADMEを英語で読め。Stack Overflowで英語で質問しろ。
それが一番の近道や。

### ② LinkedInは「最強の履歴書」

日本の就活生はリクナビ・マイナビ使うよな？
でも、**世界標準はLinkedIn**や。

俺はLinkedInのプロフィールを全部英語にして、**"Product Manager"** って勝手に名乗った。
実績なんて大したことない。「大学でアプリ作った（ユーザー数5人）」とかや。
それでも、「I built a mobile app using React Native」って書けば、それらしく見えるんや。

そしたらな、海外のリクルーターからDMが来るんや。
「Hey James, 興味ある？」って。
日本の企業みたいに「志望動機は？」なんて聞いてこーへん。
「React書ける？ OK、明日面接な」
これぐらいのスピード感や。

**ハック:**
プロフィール写真はプロに撮ってもらえ。笑顔でな。
スキル欄には流行りのキーワード（GenAI, LLM, Python）を散りばめろ。検索に引っかかるようにするんや。

### ③ 「エンジニア」ではなく「PM」を目指せ

これが一番のポイントかもしれん。
多くの人は「プログラミング勉強してエンジニアになろう」とする。
もちろんそれもええけど、これからは**AIがコードを書く時代**や。
単にコード書くだけのエンジニアは、これから価値が下がっていく可能性が高い。

でもな、**「何を作るか」「なぜ作るか」を決めるPM（プロダクトマネージャー）**の仕事は、まだAIには奪われにくい。
ビジネスとテクノロジーの橋渡しをするポジションや。

俺はコードは書けるけど、ゴリゴリのエンジニアには勝てへん。
だから、「コードも分かるPM」というポジションを取った。
これがデカい。
エンジニアと対等に話せるPMは、市場価値がめちゃくちゃ高いんや。

## 3. 具体的な年収と生活（リアルな数字）

今の俺の生活、こんな感じや。

- **年収:** $85,000（約1,275万円 ※1ドル150円換算）
- **勤務時間:** 1日6〜7時間（ミーティング2時間、作業4時間）
- **場所:** 大阪の自宅（たまにカフェ、たまに沖縄）
- **使うツール:** Slack, Notion, Jira, Zoom, GitHub Copilot

日本の同年代が満員電車に揺られて、手取り20万で残業してる間に、俺は家でコーヒー飲みながら、AIに指示出して仕事終わらせてる。
この格差、残酷やろ？

でもな、能力の差なんてほとんどないんや。
**「知ってるか知らないか」「動くか動かないか」**だけの差や。

## 4. 最後に：お前はどうする？

ここまで読んで、「自分には無理や」って思ったか？
それとも、「ちょっとやってみようかな」って思ったか？

前者なら、残念ながら日本のシステムの中で搾取され続けるしかない。
でも後者なら、お前にはチャンスがある。

まずは**LinkedInのアカウント作れ。**
プロフィールを英語で書け。
DeepL使えば一瞬やろ？

失敗しても死なん。
日本の企業に落ちても、世界には何億という企業があるんや。
視野を広げろ。世界はもっと広いし、適当で、自由や。

ほな、向こう側で待ってるで。
