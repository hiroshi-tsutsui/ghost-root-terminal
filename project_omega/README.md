# Project Omega: Japanese High School Math Visualization

A comprehensive interactive learning platform for Japanese High School Mathematics (Math I, A, II, B, III).

## Modules

### Math I: Quadratics (二次関数)
- Parabola visualization
- Vertex form exploration
- `app/quadratics`

### Math A: Probability (確率・統計)
- Normal Distribution visualization
- Standard Deviation & Mean
- `app/probability`

### Math B: Vectors (ベクトル)
- 3D Vector visualization using Three.js
- Dot Product & Cross Product
- `app/vectors`

### Math III: Calculus (微積分)
- Derivative (Tangent Line)
- Integral (Area under curve)
- `app/calculus`

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **UI**: React, Tailwind CSS
- **3D**: Three.js, React Three Fiber, Drei

## Getting Started

1. Install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)
